# 🍽️ B&M's Restaurant Website

A modern **multi-page restaurant website** built entirely with **HTML5** as part of my front-end web development learning journey. This project focuses on writing clean, semantic HTML while creating a realistic restaurant website with multiple pages, navigation, forms, and accessibility best practices.

---

## 📖 About the Project

**B&M's Restaurant** is a fictional restaurant website created to practice and improve my HTML skills. The goal was to build a complete, well-structured website similar to what a real restaurant might use.

The website includes:

- 🏠 Home
- 🍔 Menu
- 📖 About
- 🖼️ Gallery
- 💼 Careers
- 📞 Contact

---

## ✨ Features

- Semantic HTML5 structure
- Multi-page website
- Consistent navigation across all pages
- Restaurant menu with multiple food categories
- About page featuring the restaurant story and team
- Image gallery
- Careers page
- Contact form
- Embedded Google Map
- Contact information
- Internal page navigation ("Back to Top")
- Meaningful image alt text for accessibility

---

- HTML5
